# Fleet ROI Calculator (EMI)

**A professional investment calculator for fleet vehicles with EMI, resale value, and expense tracking.**

## Creator
**Shantharam G** - Franchise Intern  
WOW CARZ - Drive Your Future

## Features
- Calculate monthly EMI based on investment amount
- Annual interest rate adjustment
- Flexible tenure (1-10 years)
- Resale value estimation
- Annual expense tracking (FC & Insurance) for 2nd year onwards
- Real-time profit/loss calculation
- Responsive mobile-friendly design
